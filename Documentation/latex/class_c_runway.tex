Добавляет переданный интервал в список интервалов зоны, если он не пересекается с существующими интервалами. Проверяет, можно ли добавить переданный интервал в список уже существующих интервалов зоны без пересечений с ними. Если можно, то возвращает копию переданного интервала. Если нельзя, то рассчитывает новый, непересекающийся интервал.

{\em new\+Interval} & Интервал, который нужно проверить. \\
\hline
\end{DoxyParams}

Свободный, непересекающийся с существующими интервалами зоны, интервал. 
\end{DoxyReturn}

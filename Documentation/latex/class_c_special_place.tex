Добавляет переданный интервал в список интервалов зоны, если он не пересекается с существующими интервалами. Подразумевается, что был передан свободный (не пересекающийся интервал), который был возвращен методом \mbox{\hyperlink{class_c_special_place_a0857ed2fc03ba6a4b674661b5490b04f}{Get\+Free\+Interval()}}.

Проверяет, можно ли добавить переданный интервал в список уже существующих интервалов зоны без пересечений с ними. Если можно, то возвращает копию переданного интервала. Если нельзя, то рассчитывает новый, непересекающийся интервал.

{\em new\+Interval} & Интервал, который нужно проверить. \\

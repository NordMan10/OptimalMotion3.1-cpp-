Подробное описание

Крaткое описание класса Класс для генерации входных данных для каждого ВС. Синглтон

File : CAircraftInputDataGenerator.h Author: Yury Ten.

Длинное описание класса Генерирует входные данные для конкретного ВС, используя заранее созданные данные из классов CProgramConstants, CCommonInputData, а также данные, прописаные напрямую в методах.

Крaткое описание класса Класс для генерации входных данных для каждого ВС. Синглтон

File \+: \mbox{\hyperlink{_c_aircraft_input_data_generator_8h}{CAircraft\+Input\+Data\+Generator.\+h}} Author\+: Yury Ten.
